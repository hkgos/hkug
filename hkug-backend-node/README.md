# hkug-backend-node
Node version API server for HKUG

## To run dev
1. Clone this project
2. npm / yarn install
3. npm run dev

## To Build and run
1. Clone this project
2. npm / yarn install
3. npm run compile
4. npm start

## Usage
1. start the server
2. goto http://localhost:8080/api/topics?forum=${LIHKG/HKG}
